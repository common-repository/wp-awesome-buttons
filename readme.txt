== Installation ==

1. Install as a regular WordPress plugin
2. You have to use shortcode for generate Awesome buttons  in post or page. Example shortcode is given below.
`[btn style="green" text="Your Text"][btn style="blue" text="Your Text"][btn style="red" text="Your Text"][btn style="orange"][btn style="purple"][btn style="pink"][btn style="navy"]` or `[btn2 style="1" text="Your Text"][btn2 style="2" text="Your Text"]`
